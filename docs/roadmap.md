| ------ | ------------------------------------------------------------------------------- |
| 0      | Foundation: workspace, shells, infrastructure, tests, CI, documentation         |

| 7      | Receivables, payables, payments, retention, cash-flow forecast                  |
| 8      | Procurement, purchase orders, production, shipping, customs, warehouse receipt  |
| 9      | Installation, site logs, defects, inspection, handover                          |
| 10     | Warranty and after-sales support                                                |
| 11     | Comments, mentions, approvals, notifications, activity feeds                    |
| 12     | Marketing content, approvals, schedules, publishing queue                       |
| 13     | Email, Zalo OA, Facebook, Instagram, WhatsApp integrations                      |
| 14     | FastAPI PDF, catalog, OCR, and document-AI processing                           |
| 15     | React Native and Expo mobile application                                        |

Each sprint requires a separately approved execution plan. Later scope is documentation, not current implementation.
